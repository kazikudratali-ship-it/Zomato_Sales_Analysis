# Zomato Data Analysis

## Project Overview
This project analyzes Zomato restaurant and order data using SQL and Power BI to discover business insights and customer behavior patterns.

## Tools Used
- SQL
- Power BI
- Excel

## Key Insights
- Total Orders and Revenue
- Top Restaurants
- Order Trends
- Customer Analysis
- Delivery Performance

## Files Included
- Dataset
- SQL Queries
- Power BI Dashboard
- Dashboard Screenshots

## Project Objective
The objective of this project is to analyze restaurant sales and customer data to support business decision-making through interactive dashboards and data analysis.
